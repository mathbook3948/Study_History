## 시작하기
```shell
npx create-next-app@14
```

### 개발
- 기본적으로 React 버전 js임
#### Routing
- 기본적으로 app 폴더 + 파일 하나(page.js)를 만들면 라우팅 끝
- 폴더 이름을 주소로 추가한다
### 개발용 서버 
```shell
npm run dev
```
### 잠시 생성된 파일 보기
- page.js : 메인페이지
- layout.js : page.js 감싸는 파일
- globals.css : 모든 페이지에 적용할 css
- api 폴더 : 서버 기능 만드는 곳
